 STCR: Smart Task Conflict Resolver

A Python-based system for detecting and resolving conflicts in task scheduling, including:

- Circular dependencies among tasks
- Time overlaps for the same owner
- Deadline violations

The system prioritizes conflicts and suggests fixes automatically, making task scheduling intelligent and robust.

Features

- Detect circular dependencies** between tasks
- Detect time conflicts** between tasks assigned to the same user
- Detect deadline conflicts
- Prioritize conflicts based on severity
- Suggest possible fixes for each conflict
- Modular architecture for easy extension
